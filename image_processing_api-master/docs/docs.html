All routes

GET /all-images Get list of all image.
GET /upload Upload image UI.
POST /upload Upload image. requires FormData with option "file".
GET /public/{filename.png} Get Single image. all image.

Query Parameter

?q={quality <=100 (Default: 60)}
?s={image size}
?w={width}
?h={height}
?fit={contain, cover, fill, outside, inside}
?bg={Background color for cropped area}
